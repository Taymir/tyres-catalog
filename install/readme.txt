Инструкция по установке:
1) с помощью phpmyadmin или подобного инструмента, импортируйте файл dump.sql в вашу базу данных
2) измените в файле include/config.php настройки подключения к базе данных
3) загрузите на удаленный сервер скрипты
4) откройте в браузере файл install/generatePasses.php и введите логины с паролями для доступа к административной панели. Полученный код сохраните в файл include/admincfg.php и вновь загрузите его на удаленный сервер.
5) установите права 777 для директории upload/ и её поддиректорий.
